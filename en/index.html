Your reading list,
your server.

MyDeck is an Android client for Readeck, the self-hosted app that saves the readable content of pages you want to keep forever.

Connect it to your own Readeck server and your library stays entirely under your control — no cloud middlemen, no subscriptions, no algorithms.

🔗 Requires a self-hosted Readeck server. Getting started takes just a few minutes.

Everything you need to read without friction

Built around one idea: your reading experience should work the way you want.

☰ Pocket-style Navigation

My List, Archive, and Favorites — an interface that feels immediately familiar.

🏷 Full Label Management

Add, rename, delete, and filter by labels directly in the app — no web UI required.

↗ Quick Share

Save any link from any app via a lightweight bottom sheet with an optional auto-submit timer.

▮ Reading Progress

Per-card visual progress indicators so you always know where you left off at a glance.

🔍 In-Article Search

Full-text search with highlighted match navigation — find anything without leaving the reader.

🔼 Typography Controls

Font, size, line spacing, and content width — your reader, tuned to your taste.

🕮 Dual Reading Modes

Extracted Article view for clean reading, or Original webview when you need the full page.

🔒 Secure OAuth Login

OAuth Device Code Grant — no password stored in the app, ever.

↻ Smart Background Sync

Wi-Fi and battery-saver constraints keep your library fresh without draining your phone.

Up and reading in minutes

1
Set up Readeck
Self-host Readeck — runs anywhere Docker does, or as a standalone binary. Yours forever.
2
Install MyDeck
Download the APK from GitHub Releases, install it, and authenticate via OAuth with your server URL.
3
Save & Read
Share any link from any app into MyDeck. Bookmarks sync automatically — always ready when you are.
